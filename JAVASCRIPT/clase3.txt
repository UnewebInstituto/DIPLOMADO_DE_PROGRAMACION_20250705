JAVASCRIPT EN EL SERVIDOR

1) INSTALAR NODEJS PARA QUE PERMITA 
LA INTERPRETACIÓN DE JAVASCRIPT EN EL 
SERVIDOR.

https://nodejs.org/es

2) VERIFICACIÓN DE LAS INSTALACIONES REALIZADAS

VERIFICACIÓN DE VERSIÓN INSTALADA DE NODE
node --version 

VERIFICACIÓN DE VERSIÓN DE 
NODE PACKAGES MANAGEMENT
npm --version 

3) CREAR UN DIRECTORIO PARA 
INSTALAR LOS COMPONENTES DEL 
SERVIDOR 

cd DIRECTORIO 
cd .. SALIR DEL DIRECTORIO 
md DIRECTORIO {CREAR DIRECTORIO}

md servidor

4) CREAR UN PROYECTO NODE 
npm -y --init


// CASO TYPESCRIPT PARA POSTERIOR COMPILACIÓN

1) CREAR DIRECTORIO Y ACCEDER A ÉL
2) EJECUTAR COMANDO:
npm init -y 

3) INSTALAR DEPENDENCIA DE TYPESCRIPT 
npm install typescript --save-dev

4) CREAR ARCHIVO DE CONFIGURACIÓN DE TYPESCRIPT 
npx tsc --init 

5) CREAR ARCHIVO DE EXTENSIÓN *.TS 

6) COMPILAR ARCHIVOS
npx tsc
COMPILA TODOS LOS ARCHIVOS DE EXTENSIÓN 
*.ts A *.js

7) SE EJECUTA EL 
JS RESULTANTE
node ej01.js
