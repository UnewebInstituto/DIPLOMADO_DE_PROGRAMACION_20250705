Las interfaces de programación de aplicaciones (API) son estructuras disponibles en los lenguajes de programación que permiten a los desarrolladores crear funcionalidades complejas con mayor facilidad. Abstraen el código más complejo, proporcionando en su lugar una sintaxis más sencilla.

Como ejemplo real, piensa en el suministro eléctrico de tu casa, apartamento u otras viviendas. Si quieres usar un electrodoméstico, lo enchufas a un enchufe y funciona. No intentas conectarlo directamente a la corriente; hacerlo sería muy ineficiente y, si no eres electricista, difícil y peligroso.

De la misma manera, si quieres, por ejemplo, programar algunos gráficos 3D, es mucho más fácil hacerlo usando una API escrita en un lenguaje de nivel superior como JavaScript o Python, en lugar de intentar escribir directamente código de bajo nivel (por ejemplo, C o C++) que controle directamente la GPU de la computadora u otras funciones gráficas.

API en JavaScript del lado cliente

El JavaScript del lado del cliente, en particular, tiene muchas API disponibles: estas no son parte del lenguaje JavaScript en sí, sino que se basan en el lenguaje JavaScript central, lo que le brinda superpoderes adicionales para usar en su código JavaScript. Por lo general, se dividen en dos categorías:

  • Las API del navegador están integradas en su navegador web y pueden exponer datos del navegador y el entorno informático circundante y hacer cosas complejas útiles con ellos. Por ejemplo, la API de audio web proporciona construcciones de JavaScript para manipular el audio en el navegador: tomar una pista de audio, alterar su volumen, aplicarle efectos, etc. En segundo plano, el navegador está utilizando un código complejo de nivel inferior (por ejemplo, C++ o Rust) para realizar el procesamiento de audio real. Pero, de nuevo, la API abstrae esta complejidad.
  • Las API de terceros no están integradas en el navegador de forma predeterminada y, por lo general, debe recuperar su código e información de algún lugar de la Web. Por ejemplo, la API de Google Maps le permite hacer cosas como mostrar un mapa interactivo de su oficina en su sitio web. Proporciona un conjunto especial de construcciones que puede usar para consultar el servicio de Google Maps y devolver información específica.
