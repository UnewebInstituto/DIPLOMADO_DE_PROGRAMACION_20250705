¿Qué son las API?

Las interfaces de programación de aplicaciones (API) son estructuras disponibles en los lenguajes de programación que permiten a los desarrolladores crear funcionalidades complejas con mayor facilidad. Abstraen el código más complejo, proporcionando en su lugar una sintaxis más sencilla.

Como ejemplo real, piensa en el suministro eléctrico de tu casa, apartamento u otras viviendas. Si quieres usar un electrodoméstico, lo enchufas a un enchufe y funciona. No intentas conectarlo directamente a la corriente; hacerlo sería muy ineficiente y, si no eres electricista, difícil y peligroso.

De la misma manera, si quieres, por ejemplo, programar algunos gráficos 3D, es mucho más fácil hacerlo usando una API escrita en un lenguaje de nivel superior como JavaScript o Python, en lugar de intentar escribir directamente código de bajo nivel (por ejemplo, C o C++) que controle directamente la GPU de la computadora u otras funciones gráficas.

API en JavaScript del lado del cliente

JavaScript del lado del cliente, en particular, cuenta con numerosas API disponibles. Estas no forman parte del lenguaje JavaScript en sí, sino que se basan en el núcleo del lenguaje JavaScript, lo que proporciona superpoderes adicionales para usar en el código JavaScript. Generalmente se dividen en dos categorías:

  • Las API del navegador están integradas en el navegador web y permiten acceder a datos del navegador y del entorno informático circundante, además de realizar tareas complejas y útiles con ellos. Por ejemplo, la API de Audio Web proporciona construcciones de JavaScript para manipular el audio en el navegador: tomar una pista de audio, modificar su volumen, aplicarle efectos, etc. En segundo plano, el navegador utiliza código complejo de bajo nivel (por ejemplo, C++ o Rust) para procesar el audio. Sin embargo, la API abstrae esta complejidad.
  • Las API de terceros no están integradas en el navegador por defecto, y generalmente es necesario obtener su código e información de algún lugar de la web. Por ejemplo, la API de Google Maps permite, por ejemplo, mostrar un mapa interactivo de la oficina en el sitio web. Proporciona un conjunto especial de construcciones que se pueden usar para consultar el servicio Google Maps y obtener información específica.
