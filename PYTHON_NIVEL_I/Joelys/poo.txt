üêç Programaci√≥n Orientada a Objetos (POO) en Python
La POO se basa en el concepto de "objetos", que son instancias de "clases". Los objetos contienen datos (atributos) y c√≥digo para manipular esos datos (m√©todos).

üí° Caracter√≠sticas Fundamentales de la POO
Las cuatro caracter√≠sticas principales (pilares) que definen la POO son:

1. Abstracci√≥n
Concepto: Se enfoca en mostrar solo la informaci√≥n necesaria y esencial, ocultando los detalles de implementaci√≥n complejos. Permite definir la interfaz de un objeto sin detallar c√≥mo se implementan las funcionalidades.

Ejemplo en Python: Una clase Coche solo expone m√©todos como arrancar() o frenar(), sin que el usuario necesite saber los procesos internos del motor o los frenos.

2. Encapsulamiento
Concepto: Restringe el acceso directo a los datos internos de un objeto. Los datos (atributos) y el c√≥digo que opera sobre ellos (m√©todos) se agrupan en una unidad (la clase). Esto protege la integridad de los datos.

Ejemplo en Python: Se suelen usar convenciones de nombres (como prefijos con guiones bajos) para indicar que ciertos atributos son de uso interno y deben ser accedidos o modificados solo a trav√©s de m√©todos p√∫blicos (como getters y setters).

3. Herencia
Concepto: Permite que una clase (clase hija o subclase) herede atributos y m√©todos de otra clase (clase padre o superclase). Esto promueve la reutilizaci√≥n de c√≥digo.

Ejemplo en Python: Una clase Perro y una clase Gato pueden heredar de una clase padre com√∫n llamada Animal, obteniendo de ella m√©todos como comer() y dormir().

4. Polimorfismo
Concepto: Significa "muchas formas". Permite que objetos de diferentes clases respondan al mismo mensaje (nombre de m√©todo) de manera distinta, seg√∫n su propia implementaci√≥n.

Ejemplo en Python: Si las clases Perro y Gato (que heredan de Animal) tienen cada una su propia versi√≥n del m√©todo hacer_sonido(), al llamar a ese m√©todo en un objeto Perro sonar√° "Guau", y en un objeto Gato sonar√° "Miau".

üíª Ejemplo Pr√°ctico en Python

# Definici√≥n de la Superclase (Clase Padre)
class Animal:
    """Clase base para todos los animales."""
    def __init__(self, nombre, edad):
        # Encapsulamiento: El guion bajo simple _indica que la variable es
        # para uso interno (convenci√≥n, no estricta como en otros lenguajes)
        self._nombre = nombre
        self._edad = edad

    def comer(self):
        """M√©todo com√∫n para todos los animales."""
        print(f"{self._nombre} est√° comiendo.")

    def hacer_sonido(self):
        """M√©todo abstracto que ser√° sobrescrito (Polimorfismo)."""
        raise NotImplementedError("Las subclases deben implementar este m√©todo.")

    def get_nombre(self):
        """M√©todo 'getter' para acceder al nombre (Encapsulamiento)."""
        return self._nombre

# Definici√≥n de la Subclase (Clase Hija)
class Perro(Animal):
    """Clase que hereda de Animal."""
    def __init__(self, nombre, edad, raza):
        # Herencia: Llama al constructor de la clase padre
        super().__init__(nombre, edad)
        self.raza = raza

    # Polimorfismo: Sobrescribe el m√©todo de la clase padre
    def hacer_sonido(self):
        print(f"{self._nombre} dice: ¬°Guau guau!")

    def info_perro(self):
        print(f"Soy un {self.raza} llamado {self._nombre} y tengo {self._edad} a√±os.")

# --- Uso de los objetos ---

# 1. Crear una instancia de Perro
mi_perro = Perro("Fido", 5, "Labrador")

# 2. Uso de m√©todos heredados y propios
print(f"El perro se llama: {mi_perro.get_nombre()}")  # Uso de getter (Encapsulamiento)

mi_perro.info_perro()                                # M√©todo propio de Perro
mi_perro.hacer_sonido()                              # Polimorfismo (llama a la versi√≥n de Perro)
Resumen del Ejemplo
Herencia: Perro hereda de Animal y obtiene autom√°ticamente el m√©todo comer() y el __init__ (parcialmente).

Encapsulamiento: Se usa _nombre para indicar que el acceso debe ser a trav√©s de get_nombre().

Polimorfismo: El m√©todo hacer_sonido() en Perro se comporta de forma diferente a c√≥mo lo har√≠a en otra subclase de Animal (como un Gato).
